What is Java

Java is another High level programming language that is based on classes. It is the most popular language when looking at android smartphone applications. It was designed to be easier to use than C++ but still have the same look and feel.

Java History

Java was originally developed in 1995 by James Gosling and has been very popular ever since. Instead of compiling to straight to machine code and running, Java compiles to bytecode which is then ran by a Java Virtual Machine.
